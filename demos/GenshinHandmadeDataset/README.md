# Genshin Handmade Dataset

一个手工维护的数据集。

大胆采集，小心求证，确保每一行数据都是正确的，且不对数据源进行骚扰。（咳咳，其实是数据量太少）

⚠ 此项目真的仅供学习交流。

## Characters

| 字段名         | 含义         | 举例                                                 |
| -------------- | ------------ | ---------------------------------------------------- |
| ID             | 顺序         | `1`，表示图鉴中的第一个角色。                        |
| Rarity         | 星级         | `4`，表示四星角色。                                  |
| Name           | 名称         | `"安柏"`，角色的名称。                               |
| Title          |              | 安柏的`"飞行冠军"`、七七的`"冻冻回魂夜"`等。         |
| Element        | 元素         | `"火"`，表示角色使用的火元素。                       |
| Weapon         | 武器         | `"弓"`，表示角色使用弓箭类武器。                     |
| Gender         | 性别         | `"女"`，表示是女性角色。                             |
| Birthday       | 生日         | `"8/10"`，表示8月10号生日。                          |
| Region         | 地区         | `"蒙德"`，表示是蒙德地区的角色。                     |
| Constellation  | 命之座       | 安柏的`"小兔座"`、钟离的`"岩王帝君座"`等。           |
| HP             | 生命值上限   | `9461`，角色达到90级时的基础生命值上限（不带武器）。 |
| ATK            | 攻击力       | `223`，角色达到90级时的基础攻击力。                  |
| DEF            | 防御力       | `601`，角色达到90级时的基础防御力。                  |
| BurstEnergy    | 元素爆发能量 | `80`，角色发动元素爆发需要80点元素能量。             |
| BurstCD        | 元素爆发CD   | `20`，角色的元素爆发的技能刷新时间是20秒。           |
| LvUpBonus      | 突破提升     | 角色等级突破时额外提升的属性。                       |
| LvUpBonusDelta | 提升结果     | 角色达到90级时，等级突破所提升的对应的属性的值。     |
| Body           | 身形         | `"LOLI"`、`"GIRL"`、`"LADY"`等。                     |
